# Collaborative Study Room Platform

A full-stack real-time collaborative study room application built using the MERN Stack and Socket.IO.

## Live Demo

Frontend: https://collaborative-study-room.vercel.app

Backend: https://collaborative-study-room-lm66.onrender.com

## Features

* User Authentication (Login & Register)
* JWT Protected Routes
* Create Study Rooms
* Join Study Rooms
* Delete Study Rooms
* Real-time Chat using Socket.IO
* Active Users Display
* Typing Indicator
* Custom Study Timer
* Invite Link Sharing
* Dark / Light Theme
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Socket.IO Client

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Socket.IO
* JWT Authentication
* bcryptjs

### Deployment

* Vercel (Frontend)
* Render (Backend)
* MongoDB Atlas (Database)

## Installation

### Clone Repository

```bash
git clone https://github.com/saurabhjh007/collaborative-study-room.git
cd collaborative-study-room
```

### Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### Run Backend

```bash
cd server
npm run dev
```

### Run Frontend

```bash
cd client
npm run dev
```
